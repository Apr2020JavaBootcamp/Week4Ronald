My experience with HTML thus far

HTML is an interesting language. It is and isn't straight forward at the same time. So on one hand, HTML presents itself as very logically structured and intuitive; and it is, to a degree. Let me explain myself. Elements within HTML often have a lot of possible attributes that given the name of the attribute, you would think it did one thing, but then it doesn't; or that you have to set some attribute, that seems unrelated, before you can do anything else with the content. For example, while learning to center an image, I never would have thought I'd have to set 'display' to 'block'. Like...what? What's a block to a computer? What does that have to do with 'display'. Things like that. It's not intuitive and it feels like more of an endless list of key terms that you just have to learn. Which is ok, but it isn't my cup of tea.
