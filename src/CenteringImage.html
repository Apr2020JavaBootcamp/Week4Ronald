Centering an image with html/css

Getting an image to be centered is a simple task but it is by no means intuitive, but it makes sense once you finally pull it off. After you learn this you should be able to place your image anywhere you like. So first things first, add your image to the html doc.


Adjust the size of your image first if necessary using height and width parameters. Next in line or in css, add a 'display: block' followed by 'margin-left: auto' and 'margin-right : auto'.


That's it, your images should be centered now. Play around with the sizes. You may want to crop your images first or to make matters simpler use a 'Snipping Tool'. Snip an image from your screen and add it to an image source file. If you drag and drop it in html your IDE might set the correct size for you.
